Timetable Generator

The Timetable Generator is a simple tool that creates a timetable automatically. It takes input like teachers, classrooms, subjects, start and end times, and the day. By default, each lecture is 1 hour long. Based on the input, it generates a timetable without conflicts and displays it for you.
Features:

    Easy input of teachers, classrooms, subjects, and timings.
    Automatic timetable creation for selected days.
    Default lecture duration of 1 hour.
    Quick and hassle-free scheduling.

How to Use:

    Add the details like teachers, classrooms, subjects, and start/end times.
    Choose the day for the timetable.
    Generate and view the timetable automatically.

Perfect for schools, colleges, and organizations looking for a quick and easy way to create timetables!
